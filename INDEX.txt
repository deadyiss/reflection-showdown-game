1. reflexshowdown/            → Source code lengkap game (siap dijalankan)

   - src/                     → Server PHP (GameServer, Room, DB)
   - server.php, config.php   → Entry point & konfigurasi
   - database/schema.sql      → Skema database
   - preview.html             → Preview tampilan tanpa server
   - README.md                → Dokumentasi lengkap + cara menjalankan
   - docs/screenshots/        → Tangkapan layar sistem

2. preview.html               → (salinan) Buka langsung di browser untuk
                                 melihat seluruh tampilan & animasi game
                                 TANPA perlu menjalankan server/database.

CARA CEPAT MELIHAT GAME:
  Buka "preview.html" di browser (klik dua kali). Gunakan tombol di
  bagian atas untuk berpindah antar layar, atau tekan "Auto Tour".

CARA MENJALANKAN GAME PENUH:
  Lihat bagian "How to Run the System" di README.md.
  Ringkas: composer install → import database/schema.sql →
           php server.php → (terminal lain) php -S localhost:3000 di folder public.

CATATAN:
  - Antarmuka game & preview menggunakan Bahasa Inggris.
  - Laporan akhir menggunakan Bahasa Indonesia (format akademik).
  - Tautan di README & laporan masih placeholder — ganti dengan tautan asli.
  - Placeholder gambar pada laporan: ganti dengan tangkapan layar / diagram asli.
